inline/blocks

In CSS, the display property is used to specify the layout of an element. Two common values for the display property are "inline" and "inline-block".

Inline

"Inline" elements are placed inline with the text, and allow text to flow around them. These elements do not have a fixed width and height, and only take up as much width as necessary. They will not create a new line after the element. This paragraph is an example of using an inline element, watch what happens when I put an <img> element directly after this

Inline-block

"Inline-block" elements, on the other hand, are similar to inline elements but they can have a fixed width and height. These elements also create a new line after the element, unlike inline elements. Examples of inline-block elements include the <div> and <img> tags. This is some text written inside a inline-block element, watch the behaviour of the image following this element.

As you can see, this effect is potentially much more usable and readable on the page. As mentioned before, html conveniently builds these features into some elements so we don't have to specify this parameter ourselves.
